# Introdução

Este relatório segue as etapas do projeto de árvore de decisão, mas agora utilizando o algoritmo SVM (Support Vector Machine) para classificação do desempenho dos estudantes.

**Objetivo:**
- Explorar, analisar e preparar os dados para classificação.
- Explicar cada etapa, resultados e limitações do conjunto de dados.

O SVM é um dos algoritmos mais poderosos para problemas de classificação, especialmente quando há separação clara entre as classes. No contexto educacional, o SVM pode ser aplicado para prever o desempenho de estudantes, identificar padrões de aprovação/reprovação e apoiar decisões pedagógicas baseadas em margens ótimas de separação.

Entre as vantagens do SVM estão a capacidade de lidar com alta dimensionalidade, a flexibilidade dos kernels e a robustez contra outliers. Por outro lado, o método pode exigir maior tempo de processamento e atenção especial à escolha dos parâmetros e do kernel.

Ao longo deste relatório, cada etapa será detalhada para garantir compreensão do processo, das decisões tomadas e das limitações inerentes ao método e ao conjunto de dados utilizado.

O SVM é um algoritmo eficiente, versátil e amplamente utilizado para problemas de classificação, especialmente quando se busca precisão e robustez.
