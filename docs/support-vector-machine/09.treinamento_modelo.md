# Treinamento do Modelo SVM

O treinamento do modelo SVM é realizado após o pré-processamento e a divisão dos dados. O SVM (Support Vector Machine) é um algoritmo eficiente para classificação, especialmente em problemas com alta dimensionalidade e dados bem separados.

No contexto deste projeto, utilizamos o kernel linear para facilitar a interpretação dos resultados. O modelo é ajustado com os dados de treino e estará pronto para realizar previsões sobre o desempenho dos estudantes.

=== "Código"
    ```python
    from sklearn.svm import SVC
    svm = SVC(kernel='linear', random_state=42)
    svm.fit(X_train, y_train)
    ```
=== "Resultado"
    O modelo foi treinado com sucesso e está pronto para avaliação.
