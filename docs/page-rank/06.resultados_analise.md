## Impacto do Parâmetro d

O parâmetro de amortecimento (d) influencia diretamente o ranking dos nós. Testar diferentes valores permite analisar a sensibilidade do algoritmo:

```python
d_values = [0.60, 0.75, 0.85, 0.95]
for d in d_values:
    pr = pagerank_manual(G, d=d)
    print(f'd={d}:', sorted(pr.items(), key=lambda x: x[1], reverse=True)[:5])
```

Discussão: Valores mais baixos de d tornam o ranking mais uniforme, enquanto valores altos destacam ainda mais os hubs. O valor padrão (0,85) é recomendado para maioria dos casos.
