# PageRank no Clube de Karatê de Zachary

O clube de karatê de Zachary é um estudo clássico de redes sociais, representando as relações entre 34 membros. O PageRank permite identificar os membros mais influentes e entender a dinâmica social do grupo.

## Código

=== "Código"
    ```python
    G_karate = nx.karate_club_graph()
    G_karate_dir = G_karate.to_directed()
    pr_manual_karate = pagerank_manual(G_karate_dir, d=0.85)
    pr_nx_karate = nx.pagerank(G_karate_dir, alpha=0.85)
    ```

## Resultados

Os top 10 membros pelo PageRank são os mais centrais e influentes na rede:

=== "Código"
    ```python
    print(sorted(pr_manual_karate.items(), key=lambda x: x[1], reverse=True)[:10])
    print(sorted(pr_nx_karate.items(), key=lambda x: x[1], reverse=True)[:10])
    ```

Esses resultados podem ser comparados com outras métricas, como grau ou betweenness centrality.
