# Modelagem dos Grafos

## Grafo de Estudantes

Para construir o grafo dos estudantes, cada nó representa um aluno e as arestas indicam relações baseadas em critérios como notas iguais ou proximidade de desempenho. Isso permite analisar grupos de influência acadêmica.

=== "Código"
    ```python
    G = nx.DiGraph()
    for i, row_i in df.iterrows():
        for j, row_j in df.iterrows():
            if i != j and abs(row_i['math score'] - row_j['math score']) <= 5:
                G.add_edge(i, j)
    ```

É possível variar o critério de ligação para explorar diferentes padrões de influência, como notas de leitura ou escrita.

## Grafo do Clube de Karatê

O grafo do clube de karatê é convertido para direcionado para aplicação do PageRank. Cada nó representa um membro e as arestas indicam relações sociais.

=== "Código"
    ```python
    G_karate_dir = G_karate.to_directed()
    ```

Para visualizar a estrutura dos grafos:
=== "Código"
    ```python
    plt.figure(figsize=(8,6))
    nx.draw(G_karate_dir, with_labels=True, node_color='lightblue', edge_color='gray')
    plt.title('Rede Social do Clube de Karatê')
    plt.show()
    ```

A análise visual ajuda a identificar clusters, hubs e padrões de conectividade.
