# Carregamento dos Datasets

## Dataset de Desempenho Estudantil

O arquivo `StudentsPerformance.csv` contém informações sobre alunos, como notas em matemática, leitura e escrita, além de dados demográficos. Esse dataset é ideal para simular relações entre estudantes e identificar padrões de influência acadêmica.

```python
# Carregar o arquivo CSV
csv_path = path + "/StudentsPerformance.csv"
df = pd.read_csv(csv_path)
print(df.info())  # Verificar estrutura dos dados
print(df.describe())  # Estatísticas básicas
```

Antes de construir o grafo, é importante tratar valores nulos, normalizar scores e, se necessário, criar novas colunas para análise.

## Dataset do Clube de Karatê de Zachary

O grafo do clube de karatê de Zachary é um clássico em estudos de redes sociais, representando as relações entre membros de um clube real. O NetworkX já disponibiliza esse grafo pronto para uso.

```python
# Carregar o grafo do clube de karatê de Zachary
G_karate = nx.karate_club_graph()
print(nx.info(G_karate))  # Informações sobre o grafo
```

Esses datasets foram utilizados para construir grafos direcionados e aplicar o PageRank, permitindo comparar resultados em contextos distintos.
