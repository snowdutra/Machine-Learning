# Importação de Bibliotecas

Para realizar as análises e visualizações do PageRank, utilizamos as seguintes bibliotecas:

```python
import pandas as pd      # Manipulação e análise de dados tabulares
import numpy as np       # Operações matemáticas e vetoriais
import matplotlib.pyplot as plt  # Visualização gráfica
import networkx as nx    # Criação e manipulação de grafos
```

- **pandas**: Permite ler arquivos CSV, filtrar, agrupar e transformar dados facilmente.
- **numpy**: Usado para cálculos matemáticos, arrays e operações vetoriais, essenciais para o algoritmo PageRank.
- **matplotlib**: Gera gráficos de barras, dispersão e visualizações dos scores dos nós.
- **networkx**: Biblioteca robusta para criar grafos, calcular métricas de centralidade e visualizar redes.

Dica: Para grafos grandes, o NetworkX pode ser combinado com outras bibliotecas como igraph ou graph-tool para maior eficiência.
