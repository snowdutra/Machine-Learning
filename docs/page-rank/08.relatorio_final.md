# Relatório Final: PageRank

O PageRank é um algoritmo fundamental para análise de redes, criado por Larry Page e Sergey Brin, e amplamente utilizado para medir a relevância de páginas na web. Sua lógica se baseia na ideia de que um nó é importante se outros nós importantes apontam para ele, considerando tanto a quantidade quanto a qualidade das conexões.

Neste projeto, o objetivo foi implementar o PageRank manualmente, sem recorrer apenas a bibliotecas prontas, e comparar os resultados com a solução do NetworkX. Foram utilizados dois cenários distintos: um dataset de desempenho estudantil, simulando relações entre alunos e notas, e a rede social do clube de karatê de Zachary, referência clássica em estudos de grafos.

A implementação manual permitiu compreender profundamente o funcionamento do algoritmo, desde a inicialização dos valores até o processo iterativo de atualização dos ranks. O código foi validado por meio da comparação com o NetworkX, mostrando resultados compatíveis e reforçando a robustez da abordagem.

Os principais resultados indicam que o PageRank é eficiente para identificar os nós mais influentes em diferentes tipos de redes. No contexto educacional, foi possível destacar alunos com maior relevância, enquanto na rede social do clube de karatê, os membros centrais foram evidenciados como "hubs" de influência. O parâmetro de amortecimento (d) foi testado em diferentes valores, mostrando que o padrão (0,85) tende a estabilizar os resultados, mas variações podem alterar o ranking dos nós.

As visualizações geradas, como gráficos de barras e imagens dos grafos, facilitaram a interpretação dos resultados e permitiram uma análise comparativa entre os métodos. A documentação clara e objetiva garante que o trabalho seja reprodutível e útil para outros estudos.

Entre as limitações, destaca-se que o PageRank foi originalmente pensado para grafos direcionados, podendo exigir adaptações para redes não direcionadas. Além disso, o valor do parâmetro d pode ser ajustado conforme o contexto, mas não existe um valor universal ideal. Para trabalhos futuros, recomenda-se explorar outras métricas de centralidade, como betweenness e closeness, e aplicar o PageRank em redes maiores, de diferentes áreas, como redes acadêmicas, biológicas ou de colaboração.

Em síntese, o projeto proporcionou uma visão completa do PageRank, desde sua fundamentação teórica até a aplicação prática, reforçando a importância de entender algoritmos clássicos e suas adaptações para diferentes contextos.
